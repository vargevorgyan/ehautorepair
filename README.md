## Customization Guide

### Update Contact Information

Edit the following in `index.html`:

- **Phone Number**: Search for `(555) 123-4567` and replace with your actual number
- **Location**: Search for `123 Auto Repair Lane` and update with your address
- **Email**: Search for `info@ehautorepair.com` and replace with your email
- **Social Media**: Update the Instagram and Facebook links (search for `https://instagram.com/ehautorepair`)

### Add Real Gallery Images

Replace the placeholder gallery items in the gallery section with actual images:

```html
<div class="gallery-image" style="background-image: url('your-image.jpg');">
```

### Update Business Hours

Modify the hours in the contact section as needed.
